# DamSet 한국어 사용 설명서

DamSet은 운동 중 화면 조작을 최소화하는 iPhone용 세트·휴식 기록 앱입니다. 루틴을 미리 준비하고, 운동 중에는 실제 수행값만 빠르게 고친 뒤 세트를 완료하세요. 휴대폰을 잠그면 Live Activity가 남은 휴식 시간과 다음 세트를 이어서 보여 줍니다.

> [!NOTE]
> 현재 앱 화면의 버튼과 메뉴는 영어로 표시됩니다. 이 문서에서는 찾기 쉽도록 실제 버튼 이름을 함께 적었습니다.

## 1. 처음 실행하기

1. 앱을 처음 열면 알림 권한을 요청할 수 있습니다.
2. 잠금 화면에서 휴식 종료음을 받으려면 **알림 허용**을 선택하세요.
3. 하단에는 `Routines`와 `History` 두 탭이 있습니다.

알림을 허용하지 않아도 루틴과 기록 기능은 사용할 수 있습니다. 다만 앱을 잠그거나 백그라운드로 보냈을 때 `3·2·1` 휴식 종료 알림은 전달되지 않습니다.

## 2. 루틴 만들기

`Routines` 탭 오른쪽 위의 `+`를 누르면 새 루틴이 만들어집니다.

- **이름과 이모지**: 루틴을 구분할 이름과 아이콘을 정합니다.
- **운동 종목**: 종목 이름과 기록 방식을 선택합니다.
- **무게·횟수**: 중량 운동의 목표 무게와 반복 횟수를 입력합니다.
- **시간**: 플랭크처럼 시간으로 기록하는 운동은 목표 시간을 지정합니다.
- **휴식 시간**: 각 세트가 끝난 뒤 쉴 시간을 초 단위로 설정합니다.
- **세트**: 같은 운동을 여러 세트 추가하고 순서를 정합니다.

편집을 마친 뒤 저장하면 `My Routines`에 나타납니다. 기본으로 제공되는 Push, Pull, Legs 루틴도 자유롭게 수정할 수 있습니다.

## 3. 오늘의 운동 시작하기

1. 루틴 카드에서 `Start Workout`을 누릅니다.
2. `Today's Workout` 화면에서 오늘 수행할 운동만 켭니다.
3. 아래쪽 `Start N Sets` 버튼으로 운동을 시작합니다.

이 단계에서 운동을 빼더라도 저장된 원본 루틴은 바뀌지 않습니다. 컨디션에 따라 오늘만 일부 종목을 건너뛸 때 사용하세요.

## 4. 세트 기록하기

운동 화면에는 현재 종목, 전체 진행률, 세트 번호와 목표값이 표시됩니다.

| 조작 | 결과 |
| --- | --- |
| 횟수 `− / +` | 이번 세트의 실제 반복 횟수를 조정합니다. |
| 무게 조정 | 이번 세트의 실제 무게를 변경합니다. |
| `Set Done` | 현재 수행값을 기록하고 휴식을 시작합니다. |
| 오른쪽 위 `+` | 현재 세트를 다음 순서에 한 번 더 추가합니다. 저장된 루틴에는 영향을 주지 않습니다. |
| 왼쪽 위 `End` | 완료한 세트를 저장하거나 운동 전체를 버리고 종료합니다. |

실제 수행값은 목표보다 적거나 많아도 괜찮습니다. `Set Done`을 누른 시점의 실제 횟수·무게·시간이 최종 기록에 들어갑니다.

## 5. 휴식 타이머와 다음 세트

세트를 완료하면 루틴에 설정된 휴식 타이머가 시작됩니다.

- 남은 시간은 앱 화면과 Live Activity에서 함께 갱신됩니다.
- 휴식 중 앱을 잠가도 종료 예정 시각을 기준으로 시간이 계속 흐릅니다.
- 준비가 끝났다면 휴식을 건너뛰고 다음 세트를 바로 시작할 수 있습니다.
- 타이머가 끝나면 다음 세트로 자동 전환됩니다.
- 잘못 완료한 세트는 휴식 화면에서 수행값을 정정할 수 있습니다.

### 휴식 종료음

앱이 앞에 열려 있으면 마지막 3초에 음성 `3·2·1`과 종료음, 햅틱을 사용합니다. 운동 음악이 재생 중이라면 잠시 볼륨을 낮췄다가 복귀시킵니다.

휴대폰이 잠겨 있거나 앱이 백그라운드에 있으면 iOS 정책상 앱이 임의의 음성을 직접 재생할 수 없습니다. 대신 다음 로컬 알림을 예약합니다.

```text
3
2
1
Next set — go!
```

운동을 일찍 이어가거나 종료하면 남아 있는 휴식 알림은 취소됩니다.

## 6. 잠금 화면과 Dynamic Island

운동을 시작하면 Live Activity도 함께 시작됩니다.

- 운동 중: 현재 운동, 세트 번호, 목표 및 실제 횟수
- 휴식 중: 남은 시간과 다음 세트 재개 시각
- 지원 기기: Dynamic Island의 축약 상태
- 운동 종료: 진행 중인 Live Activity도 함께 종료

잠금 화면에서 값이 바로 바뀌지 않으면 앱을 한 번 열어 세션을 새로 동기화하세요. 알림과 Live Activity가 보이지 않는 경우 아래 문제 해결을 확인하세요.

## 7. 운동 종료와 기록

운동 화면 왼쪽 위의 `End`를 누릅니다.

- `Save Completed Sets`: 지금까지 완료한 세트만 `History`에 저장합니다.
- `Discard Workout`: 현재 운동을 저장하지 않고 종료합니다.
- `Keep Going`: 종료 창을 닫고 운동을 계속합니다.

모든 세트를 마치면 운동은 완료 상태가 됩니다. 저장된 기록에는 운동 시간, 완료 세트, 실제 수행값, 총 볼륨이 포함됩니다.

## 8. History 사용하기

`History` 탭에서 날짜별 운동 기록을 확인할 수 있습니다.

1. 운동한 날짜를 선택합니다.
2. 저장된 운동을 열어 종목별 세트를 확인합니다.
3. 잘못 입력한 무게·횟수·시간을 수정합니다.
4. 필요하면 세트 또는 운동 기록을 삭제합니다.
5. 같은 운동의 기록이 쌓이면 진행 차트에서 변화를 확인합니다.

중량 운동의 총 볼륨은 각 세트의 `실제 무게 × 실제 횟수` 합계로 계산합니다.

## 9. 알림과 Live Activity 문제 해결

### `Lock Screen 3 · 2 · 1 sound is off`가 보일 때

화면의 `Enable sounds`를 누르세요. 이미 권한 요청을 거절했다면 `Settings`를 눌러 iOS 설정에서 DamSet 알림을 허용해야 합니다.

확인할 항목:

1. 설정 → 알림 → DamSet → 알림 허용
2. 사운드 활성화
3. Focus 모드에서 DamSet 허용
4. 무음 모드와 기기 음량

### Live Activity가 보이지 않을 때

1. 설정 → DamSet에서 Live Activities가 허용되어 있는지 확인합니다.
2. 운동을 완전히 종료한 뒤 새 운동을 시작합니다.
3. 실기기 빌드라면 앱과 확장 타깃의 서명이 모두 성공했는지 확인합니다.
4. Xcode 콘솔에서 ActivityKit 시작 오류를 확인합니다.

### 음악이 멈추거나 돌아오지 않을 때

포그라운드 음성 안내는 음악을 중단하지 않고 잠시 낮추도록 설계되어 있습니다. 실제 기기와 음악 앱 조합에 따라 다르게 동작한다면 재생 앱, iOS 버전, 재현 단계를 기록하고 [QA 가이드](./qa-automation.md)의 실기기 오디오 항목을 확인하세요.

## 10. 데이터와 제한 사항

- 루틴, 진행 중 세션, 운동 기록은 기기 안의 JSON 파일로 저장합니다.
- 앱을 삭제하면 로컬 기록도 함께 사라질 수 있으므로 중요한 기록은 별도로 보관하세요.
- 무료 Apple 개발 팀으로 설치한 빌드는 프로비저닝 프로파일 만료 후 다시 빌드·설치해야 할 수 있습니다.
- 현재 iCloud 동기화, 계정, Apple Watch, iPad, macOS는 지원하지 않습니다.

개발 환경 설치와 실기기 배포는 [설치 가이드](./install.md), 기능별 검증 절차는 [QA 가이드](./qa-automation.md)를 참고하세요.
